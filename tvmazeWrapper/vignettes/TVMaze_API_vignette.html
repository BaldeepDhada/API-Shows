Using Fuctions

get_shows(): An API call is made to the TVmaze database with the television show the user inputed as a parameter. The function returns a dataframe of all television shows similar to what the user inputed, as well as all associated data found within the database. NULL is returned if the television show is not found within the database.

example:

shows <- get_shows("Game of Thrones")
head(shows, 1)

format_show_name(): This function formats the dataframe returned in the get_shows() function, returning only the television show name, premier date, end date, and genres.

example:

format_show_name(shows)

get_seasons(): second API call is made to the TVmaze database with television show id as a parameter. This function returns a dataframe of all available data pertaining to a specific television show.

format_season_name(): This function formats the dataframe returned in the get_seasons() function, returning only season numbers, season names, premier dates, end dates, and number of episodes.

example:

# seasons_df is the dataframe with seasons data
seasons <- format_season_name(seasons_df)
